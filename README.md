# 🚀 FlexTracker

Flextracker ek modern, sleek aur lightweight fitness companion hai jo aapke daily workout, calorie intake, aur protein goals ko track karne ke liye banaya gaya hai. Yeh app khas taur par un logon ke liye design ki gayi hai jo apne fitness journey ko clutter-free aur stylish tareeqay se manage karna chahte hain.

## ✨ Features

- **Dashboard:** Aaj ke goals aur progress ka instant overview.
- **Workout Logger:** Apne exercises, weight, aur reps ko easily log karein.
- **Macro Tracker:** Calories aur Protein ka hisaab rakhein.
- **Rest Timer:** Workout ke dauran rest intervals ke liye built-in timer.
- **Theme Support:** Modern aesthetic ke sath Dark/Light mode toggle.
- **Persistent Data:** Aapka data browser ki `localStorage` mein mehfooz rehta hai, refresh karne par delete nahi hota.

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS (v4)
- **Icons:** Font Awesome
- **Deployment:** Netlify
- **Build Tool:** Vite
